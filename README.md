# Easwaran's Portfolio

A minimal, research-focused portfolio built with React, Vite, and Tailwind CSS.

## Tech Stack
*   **Framework**: React (Vite)
*   **Styling**: Tailwind CSS
*   **Deployment**: GitHub Pages

## Deployment
This project is deployed using `gh-pages`.
To deploy updates:
```bash
npm run deploy
```
This will build the project to `dist/` and push it to the `gh-pages` branch.
